== Description ==


This plugin automatically hides all other shipping methods when "free shipping" is available during checkout process. It also includes an option to keep "local pickup" available alongside "free shipping"



== Installation ==



Starting with Hide Shipping Methods in WooCommerce consists of just two steps: installing and setting. Hide Shipping Methods in WooCommerce is designed to work with your site's specific needs, so don't forget to go through the Hide Shipping Methods in WooCommerce configuration wizard as explained in the 'after activation' step!

= INSTALL USER REGISTRATION FROM WITHIN WORDPRESS =
1. Visit the plugins page within your dashboard and select 'Add New';
2. Search for 'Hide Shipping Methods in WooCommerce';
3. Activate Hide Shipping Methods in WooCommerce from your Plugins page;
4. Go to 'after activation' below.

= AFTER ACTIVATION =
1. Navigate to **WooCommerce > Settings > Shipping > Shipping options**.
2. Select your option.
3. Save and enjoy!
